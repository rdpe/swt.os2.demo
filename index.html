Welcome

This is the
SwtOs2Demo Project

Besides SWT being
the Windowing System for Eclipse,
it is also used by several
other Java applications.

SwtOs2Demo is intended to show progress on SwtOs2

Developer Notes

The SwtOs2Demo
also doubles as a SWT-API test program,
(SwtApiTest), which is done by having two code-paths
internally which are easily switchable.

The SwtApiTest part
complements the Odin32 ApiTest module
for tests from the SWT-JNI level.
